# 8032 Code for the FRC 2026 Rebuilt Game

## How to contribute

To contribute to our code, you must:

* Have a solid understanding of subsystems and commands
* Be able to explain the purpose of IO layers
* Demonstrate your ability to navigate an entire robotics codebase by completing our XRP challenge
